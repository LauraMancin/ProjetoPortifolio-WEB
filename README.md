##💻 Portfólio Pessoal (Atividade em Sala)

Este projeto foi desenvolvido como uma atividade em sala de aula com o objetivo de praticar os conceitos básicos de desenvolvimento web utilizando HTML e CSS.

##📌 Sobre o Projeto

O portfólio contém duas páginas principais:

Home: apresenta uma introdução inicial.
Sobre Mim: traz informações pessoais e objetivos.

##🚀 Tecnologias Utilizadas
- HTML5
- CSS3

##🎯 Objetivo da Atividade
- Praticar a estruturação de páginas com HTML
- Aplicar estilos utilizando CSS
- Compreender a organização básica de um site

##📂 Estrutura do Projeto
/portfolio
│── index.html
│── sobre.html
│── style.css

##💡 Considerações

Este projeto representa um dos primeiros contatos com a criação de páginas web, sendo focado no aprendizado dos fundamentos.
